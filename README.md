# 俄罗斯方块

经典俄罗斯方块游戏的 Python 实现。

## 截图

![俄罗斯方块](tetris.png)

## 特性

- 完整的俄罗斯方块游戏逻辑
- 7种标准方块 + 旋转系统
- 幽灵方块预览落点
- 下一个方块预览
- 等级系统（速度随等级提升）
- 分数与消行统计
- 墙踢（Wall Kick）支持
- 暂停/重新开始功能

## 安装与运行

### 环境要求
- Python 3.7+
- pygame

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行游戏

```bash
python tetris.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格 | 直接落到底部 |
| P | 暂停/继续 |
| R | 重新开始 |

## 计分规则

- 消1行: 100 × 等级
- 消2行: 200 × 等级
- 消3行: 300 × 等级
- 消4行: 800 × 等级 (Tetris奖励)
- 硬降: 每格1分
