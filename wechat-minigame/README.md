# 微信小游戏：接星避陨石

这是一个可直接用微信开发者工具导入的小游戏项目。

## 玩法

- 左右拖动底部飞船移动。
- 接到星星加分。
- 碰到陨石扣生命。
- 右上角按钮可暂停。
- 游戏结束后点击屏幕重新开始。

## 运行

1. 打开微信开发者工具。
2. 选择“小游戏”项目。
3. 导入 `E:\test\wechat-minigame`。
4. AppID 可使用测试号或保持 `touristappid` 预览调试。

## 文件

- `game.js`：小游戏主逻辑和绘制代码。
- `game.json`：小游戏基础配置。
- `project.config.json`：微信开发者工具项目配置。
